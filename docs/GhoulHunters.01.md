Ghoulhunters is an invasive species detection and monitoring system.
Ghoulhunters — a student-designed community surveillance blueprint and prototype to help Brisbane river users spot and report freshwater gold clam (Corbicula fluminea) without moving it, supporting official biosecurity efforts.


It includes a web application that allows users to upload images of invasive species and get a report on the species to ghoulhunters.

It includes a hybrid mobile application named Ghoulhunter that allows user to capture images of invasive species and it includes the location, date, time with remarks if required and directly send the report to ghouhunters.
